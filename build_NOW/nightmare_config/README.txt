NIGHTMARE EDITION - CONFIGURATION FILES
=======================================

This directory contains configuration files for Nightmare Edition features.
Create empty files with these names to enable features:

  enable_debug_features - Enable feature 8
  enable_visual_effects - Enable feature 7
  enable_enhanced_audio - Enable feature 6
  enable_nightmare_quests - Enable feature 5
  enable_secret_levels - Enable feature 4
  enable_nightmare_ambience - Enable feature 3
  enable_nightmare_items - Enable feature 2
  enable_post_diablo - Enable feature 1
  enable_nightmare_lighting - Enable feature 0

Feature descriptions:
  enable_nightmare_lighting - Atmospheric lighting effects
  enable_post_diablo - Content available after defeating Diablo
  enable_nightmare_items - Unique nightmare-themed items
  enable_nightmare_ambience - Renamed NPCs, places, darker atmosphere
  enable_secret_levels - Additional secret levels (Level 17: The Void)
  enable_nightmare_quests - Post-Diablo quests and challenges
  enable_enhanced_audio - Advanced audio effects and atmosphere
  enable_visual_effects - Enhanced visual effects and shaders
  enable_debug_features - Debug and testing features

Example: To enable atmospheric lighting, create an empty file named 'enable_nightmare_lighting'
The file content doesn't matter - only its existence is checked.
